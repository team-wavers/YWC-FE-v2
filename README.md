## 전남청년 문화복지카드 가맹점 검색 서비스 v2

### 🔎 서비스 개요

전라남도에서 시행하는 문화복지카드 사업의 결제 가능 가맹점을 검색할 수 있는 서비스 입니다.
지도를 통해 현재 위치 주변의 가맹점 및 키워드 검색을 통해 가맹점 위치를 파악할 수 있습니다.
